# Nu Kenzie

Aplicação desenvolvida para gerenciar finanças pessoais, possibilitando o controle de entradas e saídas de valores, além de calcular o total disponível.
## Tecnologias Utilizadas

- React
- JavaScript
- Vite
- CSS Modules
- HTML5
- Responsividade com Flexbox
- Deploy: Vercel

## Funcionalidades

- Cadastrar novas transações (entradas ou saídas)
- Listar todas as transações cadastradas
- Exibir o total disponível somando todas as transações
- Excluir transações individualmente
- Página inicial com botão "Iniciar" para entrar na aplicação
- Voltar para a landing page a partir do header
- Feedback visual com SVG quando não há transações cadastradas
- Layout fiel ao Figma
- Aplicação 100% responsiva
